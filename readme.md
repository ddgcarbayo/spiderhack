# SpiderHack
### Slideshare to pdf converter 

Only for NodeJS 8+ versions

#### Binary command:

```sh
$ npm install -g spiderhack
$ spiderhack
```
#### Or clone

```sh
$ git clone git@github.com:ddgcarbayo/spiderhack.git
$ cd spiderhack
$ npm install
$ npm start
```
